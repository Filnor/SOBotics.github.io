SE have provided us with a feed and allows testing, however stating clearly We do request that you don't use this as a witchhunt to find users and harass them, to comply users in chat needs to follow these rules:

  1. Commenting back on reports is strictly prohibited. Users found leaving comments on the SO post when acting on reports will be kicked from SOCVFinder chat room, and RO's should terminate the comment feed until issue is not solved using the command @Queen stop feed.
  2. It is not allowed to speak about the users involved in SOCVFinder chat room. We do not moderate users, but comments.
  3. When multiple offensive comments are exchanged, flag the thread for moderator intervention and move on.
  4. tp should be used for "not nice" comments or comments that lead to offensive ones or offensive / spam posts (answer / question). You can flag the comment or the post approprietly.
  5. tp socvr can be used to send a notification to the SOCVR room. This should only be used when the comment is very rude and requires immediate attention to avoid flam posts and / or repeatingly offense from other users.
  6. fp should be used for comments that were wrongly reported (i.e. they are on-topic and constructive).
  7. It is up you to give feedback when and if you want to. You can abstain from giving one if you feel the comment is hard to classify (borderline).
